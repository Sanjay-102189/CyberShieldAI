# CyberShieldAI

AI-Powered Cyber Threat Analyzer Mobile Application

## Overview

CyberShieldAI is an AI-driven cybersecurity mobile application that helps users detect phishing URLs, suspicious messages, malicious files, and security threats using advanced AI analysis.

## Features

* URL Threat Scanner
* Message Analyzer
* File Analyzer
* AI Security Assistant
* Threat Intelligence Dashboard
* Scan History
* Threat Reports

## Technology Stack

* Flutter Mobile App
* Node.js Backend
* MongoDB Atlas
* Firebase Authentication
* Gemini AI API
* Groq API
